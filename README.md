# UAE-MPCI 知识库

阿联酋海运预装货物信息计划（Maritime Pre-Load Cargo Information）中文综合查询系统。

## 关于 MPCI

MPCI 是由阿联酋国家预申报信息中心（NAIC）领导的政府计划，要求船公司和货运代理在货物装载前向 NAIC 提交提单数据。该计划于 2025 年 6 月正式生效。
